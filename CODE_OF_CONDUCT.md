# Code of Conduct

Worldweft welcomes contributors building thoughtful, testable agent worlds.
Treat other contributors with respect, keep criticism technical and specific,
and assume good faith.

Harassment, personal attacks, discriminatory language, doxxing, and sharing
credentials or private data are not acceptable. Maintainers may remove content,
close discussions, or block participation when needed to protect the project.

For sensitive conduct concerns, contact the maintainer through GitHub rather
than opening a public issue.
